# Nvwa-2

Nvwa-2 is a game project developed in Python. The project features a graphical user interface and various game assets, including images and GIFs for events and outcomes.

## Features
- Interactive gameplay with random events
- Visual assets for game endings and events
- Dice mechanics
- Organized static resources

## Project Structure
```
main.py           # Main entry point of the game
static/           # Static resources
  UI/             # User interface assets
  游戏结局png/     # Game ending images (PNG)
  随机事件gif/     # Random event GIFs
  骰子/           # Dice images/assets
```

## Requirements
- Python 3.x
- (Optional) Additional Python packages for GUI (e.g., tkinter, PyQt, etc.)

## How to Run
1. Make sure you have Python 3 installed.
2. Run the main script:
   ```powershell
   python main.py
   ```
